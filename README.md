# Achilleus Docs

Documentation for Achilleus — a security monitoring platform for developers managing multiple client websites.

Achilleus scans domains across SSL/TLS, security headers, and DNS/email configuration, then surfaces findings in a dashboard with composite scores and professional PDF reports. Built for freelancers and small agencies who need visibility without enterprise complexity.
